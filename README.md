Terraform GKE Cluster
=====================

## Description

This is a simple project to create a GCP Kubernetes Cluster with Terraform

## How does it work

Install terraform: https://learn.hashicorp.com/tutorials/terraform/install-cli

Run those following command:

    terraform init
    terraform plan
    terraform apply
